# 💪 Pull-Ups Counter

An AI-powered Pull-Ups Counter built using **Python, OpenCV, and MediaPipe Pose**.
This project detects human body landmarks in real time through a webcam and counts pull-ups based on vertical movement.

---

## 🚀 Features

* 🎯 Real-time pose detection
* 🔢 Automatic pull-up counting
* 📷 Webcam-based tracking
* ⚡ Smooth and lightweight performance
* 🧠 AI-based body landmark tracking

---

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/rsamwilson2323-cloud/pullups-counter.git
cd pullups-counter
```

2. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the program:

```
python pullups_counter.py
```

* Stand in front of the camera
* Perform pull-ups
* Counter will display near your head

👉 Press **ENTER** to exit

---

## ⚠️ Requirements

* Python 3.8 – 3.11
* Webcam
* Good lighting for better accuracy

---

## 📸 How It Works

* Detects body using **MediaPipe Pose**
* Tracks nose position
* Counts rep when moving from **down → up**
* Displays count dynamically on screen
